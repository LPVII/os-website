Alfred is a useful macOS app, which acts like the spotlight tool, but a lot more powerful. One of the best features about this app is its extensibility. Many designers, developers and productive people have taken advantage of this and created some useful plugins for it.

&nbsp;&nbsp;
-----

Here's some basic features of alfred, if you haven't seen it yet. It's only on mac unfortunately, but worth checking out. There's plenty of resources available going over the basics of it. Not as many posts about peoples curated setups.

&nbsp;&nbsp;
-----

## The Plugins

&nbsp;&nbsp;
-----

For anyone who has it, or is playing around with it - I'm going to keep a list of any useful plugins existing related to web development or general tooling.

&nbsp;&nbsp;
-----

### Web Development

&nbsp;&nbsp;
-----

- Base64 Image
  - base64 encode/decode an image
  - <a href="https://github.com/skynebula/base64-alfred-workflow" target="_blank">Link</a>
- Color Codes
  - convert colour codes to any format
  - <a href="http://www.packal.org/workflow/colors" target="_blank">Link</a>
- Dev Docs Reference (<a href="https://devdocs.io" target="_blank">devdocs.io</a>)
  - reference anything on this vast dev documentation site
  - <a href="https://github.com/yannickglt/alfred-devdocs" target="_blank">Link</a>
- Dev Hints Reference (<a href="https://devhints.io" target="_blank">devhints.io</a>)
  - reference anything on this vast dev cheat sheet site
  - <a href="https://github.com/kevinjalbert/alfred-devhints" target="_blank">Link</a>
- Npms
  - search for npm packages
  - <a href="https://github.com/sindresorhus/alfred-npms" target="_blank">Link</a>
- Placeholder Images
  - generate a placeholder image to use for mockups
  - <a href="https://github.com/matuzo/alfred-lorempixel-workflow" target="_blank">Link</a>
- Cd Terminal
  - open a folder in terminal
  - <a href="https://github.com/janclarin/open-in-terminal-alfred" target="_blank">Link</a>
- Tailwind Docs
  - reference tailwind docs
  - <a href="https://github.com/techouse/alfred-tailwindcss-docs" target="_blank">Link</a>
- VSCode Open Folder
  - open a folder in vscode
  - <a href="https://github.com/alexchantastic/alfred-open-with-vscode-workflow" target="_blank">Link</a>

### Tools

&nbsp;&nbsp;
-----

- About This Mac
  - get info about mac hardware & software
  - <a href="https://github.com/xilopaint/alfred-about-this-mac" target="_blank">Link</a>
- Number Coverter
  - convert different number formats (currency, temp etc.)
  - <a href="https://github.com/deanishe/alfred-convert" target="_blank">Link</a>
- Generate uuid
  - generate a random, unique id
  - <a href="https://github.com/MariusRumpf/alfred-uuid" target="_blank">Link</a>
- Markdown Table
  - generate the formatting for a markdown table
  - <a href="https://github.com/crispgm/alfred-markdown-table" target="_blank">Link</a>
- Search Notes
  - search apple notes directly in Alfred
  - <a href="https://github.com/sballin/alfred-search-notes-app" target="_blank">Link</a>
- Shorten URL
  - shorten a url using some popular shortening services
  - <a href="https://github.com/ilstar/Shorten-URL" target="_blank">Link</a>
- Weather
  - get the local weather forecast
  - <a href="https://github.com/jason0x43/alfred-weather" target="_blank">Link</a>
